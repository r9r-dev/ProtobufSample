# Protobuf Sample
.Net Protobuf Client/Server Sample

## Client
Simple Console client

## Server
Simple server using gRPC and Topshelf for self hosting in Windows as a service.

## Prerequistes
- Visual Studio 2017
- Target for .Net Framework 4.7.1

logs are saved in logs.log using nlog
